La vie d'Ada

Ada Lovelace, mathématicienne britanique du XIXe siècle, est considérée comme la première programmeuse de l'histoire pour avoir écrit le premier alghorithme destiné a être éxécuter par une machine, la machine analytique de Charles Babbage, ce qui en fait une pionnière visionnaire de l'informatique moderne.

Le nom de l'école Ada Tech School s'inspire de cette mathématicienne.

En quelques points :

  • Pionnière de l'informatique : créa le premier alghorithme pour ordinateur
  • Mathématicienne visionnaire du 19ème siècle
